Parameters
plateauL'objet Plateau contenant les données du plateau de jeu à afficher.

Cette méthode dessine le plateau de jeu dans le GridPane, en utilisant différentes couleurs pour représenter les types de cases. Chaque case est représentée par un Rectangle de JavaFX.

Demande à l'utilisateur le nombre de joueurs participant au jeu.

Returns
Le nombre de joueurs choisi par l'utilisateur.

Affiche une boîte de dialogue demandant le nombre de joueurs. La méthode valide l'entrée de l'utilisateur et répète la demande jusqu'à ce qu'un nombre valide soit entré. Si l'utilisateur annule la boîte de dialogue, l'application se ferme.
